# PRD.md

## 1. 제품명
감성 분석 웹 서비스

## 2. 제품 목적
사용자가 입력한 문장을 AI가 분석하여 긍정, 부정, 중립 중 하나의 감성 결과를 제공한다. 비전공자도 쉽게 사용할 수 있는 간단한 웹 서비스로 만든다.

## 3. 핵심 사용자
- 짧은 글의 분위기를 빠르게 확인하고 싶은 사용자
- 고객 리뷰, 설문 응답, 댓글의 감성을 확인하고 싶은 사용자
- AI 서비스 구조를 학습하는 교육생

## 4. 핵심 기능
1. 텍스트 입력
2. 감성 분석 요청
3. 긍정/부정/중립 결과 표시
4. 신뢰도 백분율 표시
5. 분석 이유 표시
6. 오류 메시지 표시
7. Supabase에 분석 이력 저장

## 5. 구현 범위
이번 버전에서는 단일 사용자가 웹페이지에서 텍스트를 입력하고 결과를 확인하는 기능만 구현한다.

## 6. 제외 범위
- 회원가입/로그인
- 관리자 페이지
- 결제 기능
- 대량 파일 업로드
- 다국어 UI 전환
- 모바일 앱 출시

## 7. 기술 스택
- Front-end: HTML, CSS, JavaScript
- Back-end: Node.js
- AI: OpenAI API
- Database: Supabase
- Deployment: Vercel

## 8. 성공 기준
- 사용자가 문장을 입력하고 감성 결과를 확인할 수 있다.
- 결과는 긍정/부정/중립 중 하나로 표시된다.
- 신뢰도는 0~100 사이 숫자로 표시된다.
- 분석 이유는 한두 문장으로 표시된다.
- 오류 상황에서 사용자가 이해할 수 있는 안내가 나온다.

## 9. 검증 기준
- 빈 입력값 테스트
- 정상 문장 입력 테스트
- OpenAI API 오류 테스트
- Supabase 저장 성공 테스트
- Supabase 저장 실패 테스트
- Vercel 배포 후 동작 테스트
