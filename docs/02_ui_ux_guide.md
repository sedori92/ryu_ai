# 02_ui_ux_guide.md

## UI 참고 이미지 방향
첨부 이미지는 따뜻한 갈색 배경, 부드러운 조명, 새와 사람의 시선이 마주하는 감성적인 분위기를 가진다. 이 이미지를 그대로 복제하지 말고 다음 분위기만 반영한다.

## 디자인 키워드
- 따뜻함
- 차분함
- 집중감
- 카드형 입력 영역
- 넓은 여백
- 부드러운 색감
- 감성적인 서비스 느낌

## 화면 구성
1. 전체 배경
   - 따뜻한 브라운 또는 베이지 계열 그라데이션
   - 화면 중앙에 주요 카드 배치

2. 헤더 영역
   - 서비스명: “AI 감성 분석기”
   - 설명 문구: “문장 속 감정을 긍정, 부정, 중립으로 분석합니다.”

3. 입력 카드
   - textarea
   - placeholder: “분석할 문장을 입력하세요.”
   - 분석 버튼

4. 결과 카드
   - 감성 결과 배지
   - 신뢰도 백분율
   - 분석 이유

5. 오류 메시지 영역
   - 빈 입력 또는 API 오류 시 표시

## 반응형 기준
- PC에서는 카드 최대 너비 720px
- 모바일에서는 좌우 여백 16px
- 버튼은 모바일에서 전체 너비 사용

## UI 완료 기준
- 입력 영역이 명확히 보인다.
- 분석 버튼이 쉽게 찾을 수 있다.
- 결과가 한눈에 구분된다.
- 긍정/부정/중립 상태가 시각적으로 구분된다.
